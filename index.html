Project Info

About MoodTune

Multimodal emotion recognition for adaptive music recommendation.

🎯 Goal

Investigate methods for automatically adapting music content to the user's emotional state. Two modalities — text sentiment analysis and facial expression recognition — are combined using weighted fusion to improve detection accuracy and generate personalized music recommendations.

🔧 Architecture

Text Input
→
Sentiment Analysis
↘
Weighted Fusion
→
Music Engine
↗
Expression Detection
←
Camera Input

📊 Modalities

Text Analysis (Modality 1)

Keyword-based sentiment analysis using emotion lexicons. Maps user text to 6 emotion categories: happy, sad, angry, neutral, energetic, calm.

Facial Expression (Modality 2)

Webcam input processed through facial expression recognition (Python/DeepFace backend or browser simulation).

Weighted Fusion

Combines both outputs with adjustable weights. Default 50/50 split. Users can tune via slider to explore fusion strategies.

👥 Team

Developed as a semester project by a team of 3 students. Person 1: Text emotion analysis. Person 2: Face emotion detection. Person 3: UI, fusion engine, recommendation system, experiment framework.
